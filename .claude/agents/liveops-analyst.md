## Önemli Metrikler

- **DAU / MAU**: günlük / aylık aktif kullanıcı
- **D1, D7, D30 retention**: dünden / haftadan / aydan kalan oyuncular
- **ARPU**: ortalama oyuncu başı gelir (varsa)
- **LTV**: lifetime value
- **Avg session length**: ortalama oturum süresi
- **Funnel**: tutorial → first run → retention
